# Font_For_Microkontroller

Готовые русские шрифты для микроконтроллеров. Несколько шрифтов разных размеров. В шрифты уже вшита программа для их отображения. 
Этой программе нужно только функции отоброжения одного пикселя по координатам X Y. Свои функции нужно подставить с .c файл, заменив внутри функции Char+название шрифта (на примар CharNweCalibriLight14) функцию Pixel() на свою функцию отображения пикселя с по координатам с указанием цвета (к примеру ssd1306_drowpixel(x, y, color)).
Если в функции Char+название шрифта или String+название шрифта передать BkgColor 0x02, то фон будет "прозрачным".

Функция выводит шрифт переменной ширины (в зависимости от ширины символа).
Это не самый быстрый способ вывода шрифта, за-то универсальный.
Так же в пакете есть генератор шрифтов FontCvt и программа на C, для обработки сгенерированного шрифта. Программа не доведена до идеала, по этому работает только из dev c++. Там необходимо указать адрес сгенерированного файла и запустить выполнение в режиме отладки. После завершения будут сформированы два файла .c и .h. В этих файлах будет и шрифт и программа для его отображения с помощью функций вывода одного пикселя по координатам.
